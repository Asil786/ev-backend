
This backend supports BOTH env formats:

MYSQL_DB / MYSQL_USER / MYSQL_PASSWORD / MYSQL_HOST / MYSQL_PORT
OR
DB_NAME / DB_USER / DB_PASSWORD / DB_HOST / DB_PORT

You only need ONE set.

Example .env:

MYSQL_DB=chargup
MYSQL_USER=root
MYSQL_PASSWORD=******
MYSQL_HOST=localhost
MYSQL_PORT=3306
PORT=4000
